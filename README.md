# ScholarHub

An academic platform designed for researchers and students to discover, share, and accurately cite study materials and publications. 

## Key Features
* **Secure Authentication:** Institutional access secured via Google OAuth integration.
* **Knowledge Discovery:** A global search engine for filtering materials by subject, title, and description.
* **Impact Tracking:** Automated continuous milestone logic to track citations and calculate researcher h-indexes.
* **Platform Integrity:** Built-in moderation hub for reporting and managing academic resources.
